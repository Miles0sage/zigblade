# ZigBlade 🗡️

The first pocket-sized **offensive** security testing tool for Zigbee, Thread, and Matter protocols.

Built on ESP32-H2. KillerBee reborn on modern hardware.

> POOM lets you SEE Zigbee traffic. ZigBlade lets you BREAK Zigbee networks.

## Status: Phase 1 — MVP Development

## Hardware
- ESP32-H2-DevKitM-1 ($10)
- 1.3" SH1106 OLED I2C 128x64
- 4x tactile buttons
- MicroSD breakout
- 3.7V 1000mAh LiPo + TP4056
